# Python Logic & Algorithmic Collection 🚀

A comprehensive repository of Python scripts and mini-projects ranging from foundational mathematical logic to advanced algorithms and data structures. This collection serves as a practical playground for mastering Python syntax, optimizing time/space complexity, and building problem-solving muscle.

## 📌 Project Architecture

This repository is divided into four primary sections, scaling in complexity from raw logic to advanced system architecture.

### Part 1: Special Number Logic
Focuses on mathematical operations, loops, and condition checks.
- [x] **Magic Number:** Repeated digit summation logic.
- [x] **Tech Number:** Digit splitting and exponential math.
- [x] **Neon Number:** Square and digit summation equivalence.
- [x] **Disarium Number:** Positional power calculations.
- [x] **Harshad (Niven) Number:** Modulo and divisibility logic.
- [x] **Automorphic Number:** Trailing digit validation.

- [x] **Fascinating Number:** String concatenation and exact-match validation.
- [x] **Kaprekar Number:** Squared splitting and summation.
- [x] **Tribonacci Generator:** Advanced sequence generation.

### Part 2: String Manipulation & Creative Logic

- [x] **The Vowel Eater:** Iteration and conditional string mutation.
